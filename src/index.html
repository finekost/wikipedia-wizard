A Wikipedia list to JSON converter

For anyone in need of some structured data

Wiki wizzard can parse several wikipedia pages and transform the data into json.

You need a list of all the US Presidents or all countries with capitals? Wiki wizzard parses the wikipedia page and generates a JSON file for you.

You need a feature?

I'll do my best to add it.

Whether you added a list, improved the code or just have a suggestion for a feature. You are more than welcome to contribute.

As a coder, please use github. Otherwise just send me a tweet.

Based on the great work of others.

This project relies heavily on other peoples work.

My parser transforms the results from wtf_wikipedia an awesome javascript wikipedia markup parser. By Spencer Kelly

The frontend was styled entirely with tailwindcss an utility-first CSS framework. By Adam Wathan

I used svg icons from heroicons-ui. A very useful iconset by Steve Schoger. He provides some awesome design tips aswell. You should check out his website.
